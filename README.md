REGISTRO DI CLASSE
========

*Registro di classe* è un software per la gestione dei voti scolastici
progettato per [Madisoft][1].

```sh
$ git clone https://github.com/Carteni/madisoft-test.git
$ cd madisoft-test
$ composer install
$ make fixtures
$ npm install
$ make
$ make test
```
